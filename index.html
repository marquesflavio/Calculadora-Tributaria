O Simples Nacional

Atividades profissionais intelectuais tem uma maior carga de impostos no regime tributário do Simples Nacional. Porém existe a possibilidade de utilização de uma modalidade de redução do percentual de imposto aplicado sobre a receita.

O Simples Nacional possui 5 anexos "tabelas" de alíquotas de impostos, uma para cada tipo de atividade, e as atividades intelectuais, como a atividade de desenvolvimento de softwares está num desses grupos. Para a nossa infelicidade é no grupo mais caro.

Porém agora vamos descobrir uma possibilidade lícita de diminuir esse percentual de imposto de uma forma prática e direta.
